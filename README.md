# MyBlog 指南

## 概览

技术栈

- Node.js 
- TypeScript(还没上)
- webpack/gulp
- Promise 
- MongoDB
- es6

快速运行

```bash
$ node -v
v4.3.2
$ npm config set registry http://registry.npm.taobao.org/
$ npm config set disturl http://npm.taobao.org/mirrors/node
$ git clone git@github.com:luruozhou/MyBlog.git
$ cd MyBlog
$ npm install
$ gulp build 
$ npm run build 
```
